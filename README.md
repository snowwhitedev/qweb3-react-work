## Local Development

- Clone repo\
  `git clone https://github.com/snowwhitedev/qweb3-react-work.git`

- Install top-level dependencies\
  `yarn`

- Install sub-dependencies\
  `yarn bootstrap`

- Build and watch for changes\
  `yarn start`
